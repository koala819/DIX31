<text>
  Le plugin "Pause Click" est idéal pour améliorer votre expérience de
  visionnage sur VLC. Avec ce plugin, vous pouvez facilement mettre en pause ou
  reprendre une vidéo en cliquant directement sur l'image, ce qui est
  particulièrement utile si vous regardez des vidéos tout en effectuant d'autres
  tâches sur votre ordinateur.
</text>

<h3>Installation du plugin sur Windows</h3>

<ol className='list-decimal'>
  <li>
    <div>**Téléchargez le plugin** : Rendez-vous sur <Link href='https://github.com/nurupo/vlc-pause-click-plugin#windows' className='hover:underline' target='_blank'> le repo GitHub "vlc-pause-click-plugin"</Link> et télécharger la dernière version pour Windows.</div>
  </li>

  <li>
    **Extraction de l'archive** : Une fois téléchargée, extraire l'archive dans le dossier des plugins de VLC. <b>C:\Program Files\VideoLAN\VLC\plugins\video_filter</b>
  </li>
</ol>

<h3>Configuration et utilisation</h3>

<ol className="list-decimal">
  <li>
    <div>
      <strong>Activer le plugin</strong> : Ouvrir VLC et aller dans{' '}
      <b>Outils -> Préférences</b>. Changer le mode des réglages en bas à gauche
      en sélectionnant <b>Tous</b>.
    </div>
    <ol>
      <li>
        <div>
          <strong>Activer le plugin dans les interfaces de contrôle</strong> :
        </div>
      </li>
      <li>
        Naviguez jusqu'à <b>Interface -> Interfaces de contrôle</b> et cocher
        l'option <b>Pause/Play vidéo sur clic de souris</b>.
      </li>
      <li>
        Rendez-vous dans <b>Vidéo -> Filtres</b> et cocher également{' '}
        <b>Pause/Play vidéo sur clic de souris</b>.
      </li>
    </ol>
  </li>
</ol>

<h3>Conclusion</h3>

<text>
  Avec le plugin "Pause Click", vous pouvez contrôler VLC d'une manière plus
  intuitive et pratique. L'installation et la configuration sont simples, même
  pour les utilisateurs moins expérimentés. Essayez-le et améliorez votre
  expérience VLC dès aujourd'hui !
</text>
