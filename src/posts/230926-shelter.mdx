Admettons-le, la plupart d'entre nous utilisent des applications intrusives qui
collectent nos données personnelles sur nos smartphones. Que ce soient les
réseaux sociaux, les jeux ou la messagerie professionnelle, il existe une
solution pour limiter ces intrusions : isoler l'exécution de ces applications
dans un profil professionnel sur les appareils Android. Cette séparation permet
de garder les données et les applications professionnelles distinctes des
données personnelles. Les applications cloisonnées dans le profil professionnel
n'ont plus accès aux contacts, calendriers, ou photos du profil principal. De
plus, vous pouvez geler l'exécution de ces applications et les réactiver
uniquement lorsque vous en avez besoin, ce qui contribue à économiser la
batterie et à préserver vos données. Tout cela est possible grâce à
l'application gratuite [**Shelter**](https://github.com/PeterCxy/Shelter), que
j'utilise depuis plusieurs mois et que je pense mérite d'être mieux connue du
grand public.
