<h3> Étape 1 : Comprendre GitHub</h3>

GitHub est une plateforme où les développeurs stockent et gèrent leur code. Pensez-y comme une bibliothèque numérique où chaque livre est un projet (ou "repository", souvent abrégé en "repo") contenant tout le code nécessaire.

<h3> Étape 2 : Installer Git</h3>
<ol className='list-decimal'>
  <li>
    <div>**Téléchargez Git** : Rendez-vous sur <Link href='https://git-scm.com/downloads' target='_blank'>git-scm.com</Link> et téléchargez la dernière version pour Windows.</div>
</li>

  <li>
    **Installez Git** : Ouvrez le fichier téléchargé et suivez les instructions.
    Laissez les options par défaut si vous n'êtes pas sûr.
  </li>
</ol>

<h3> Étape 3 : Cloner le Repo</h3>
<ol className='list-decimal'>
  <li>
    <strong>Ouvrez le terminal</strong> : Appuyez sur les touches 'Win + R',
    tapez 'cmd' et appuyez sur la touche 'Enter'.
  </li>
  <li>
    <div className='flex'>
      **Clonez le Repo** : Allez sur la page <Link href='https://github.com/koala819/StarterPack' target='_blank'> GitHub du repo "StarterPack"</Link>, cliquez sur le bouton vert "Code" et copiez l'URL. Dans le terminal, tapez
      :
    </div>

    ```shell

git clone https://github.com/koala819/StarterPack.git

```
  </li>
</ol>
