<h4>2. Supprimer les branches locales</h4>

Pour supprimer une branche locale, nous pouvons utiliser la commande git-branch avec l'option -d ou -D.

<div className='text-center'>
  <Snippet color='primary'>git branch (-d | -D) < branchname ></Snippet>
</div>

- L'option -d est un alias de --delete, qui supprime la branche uniquement si elle a été entièrement fusionnée dans sa branche source.
- L'option -D est un alias de --delete --force, qui permet de supprimer la branche indépendamment de son statut de fusion

<h4>2. Supprimer les branches distantes</h4>

Pour supprimer les branches distantes à l'aide de la commande git-branch, spécifions l'option -r avec l'option -d.

<div className='text-center'>
  <Snippet color='primary'>git branch -d -r origin/< branchname ></Snippet>
</div>
